# Skill: IEC 62304 Reference

This Skill bundle is loaded on demand by agents that need authoritative
IEC 62304:2006+A1:2015 references. Loading is governed by `BaseAgent`
which lazily reads the relevant section into the system prompt.

## Files

- `clauses.md` — section-by-section summary of clauses 5.1 through 9.8
- `class_c_modules.md` — list of MSTool-AI Class C modules with their hazards
- `evidence_rubric.md` — what counts as strong / adequate / weak / missing
- `req_id_format.md` — naming convention and category mapping for REQ-IDs
